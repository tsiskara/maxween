## Changes (all in `boost.html`, ~8 surgical edits)

### 1. Core math — `CFG` + `crashFromHash`
- `HOUSE_EDGE: 0.03 → 0.01` (matches Stake/Aviator; RTP 97%→99%; instant-bust 3%→1% → far fewer brutal 1.00× busts → longer sessions)
- Add `MAX_MULT: 1000` (kills 38s boring mega-rounds + float-precision risk; P(crash≥1000)≈0.1% so the moon-shot dream stays alive)
- Cap result inside `crashFromHash` so provably-fair verification stays consistent
- **Pacing**: `BET_TIME: 6000 → 5000`, `CRASH_PAUSE: 3800 → 3000` (matches Aviator/Stake cadence)
- **Perfect-landing rule**: if a round actually reaches the 1000× cap (≈1-in-1000), auto-cash all remaining bets/bots at the cap before resolving — a jackpot moment instead of a punishment

### 2. Live bettors table (the big feature) — upgrade existing bot system
Convert `#liveFeed` from a post-round fading feed into a **live, persistent round table**:
- **BET phase**: `botRoundStart()` already assigns bets → now also render each `in` bot as a row immediately (name + bet + 🟡 status). Include the player's own bets tagged as "you".
- **FLY phase**: add `botMaybeCash(mult)` called every frame — bots whose `cashAt` is reached light up **green in real time** with their multiplier + winnings, exactly like Aviator's live cash-out feed. This is the social-proof engine.
- **CRASH**: remaining bots bust (rows red), then clear for next round.
- Keeps existing `#chatBar` as-is.
